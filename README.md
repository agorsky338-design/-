# ПРРК «Интерьер» — предрелизная сборка

## Что внутри
- `index.html` — основной сайт.
- `privacy.html` — политика обработки данных.
- `send.php` — обработчик формы через PHP `mail()`.
- `assets/` — стили, JavaScript, фотографии и PDF.
- `.htaccess` — базовые настройки Apache.

## Быстрый запуск локально
Откройте `index.html` напрямую или запустите локальный сервер:

```bash
python -m http.server 8080
```

## Публикация
1. Сделайте резервную копию текущего сайта.
2. Загрузите содержимое этой папки в корень тестового поддомена.
3. Проверьте сайт на телефоне и компьютере.
4. Проверьте, работает ли `send.php`. На сервере должна быть настроена PHP-функция `mail()`.
5. Подтвердите адрес получателя в `send.php` (`info@prrk.ru`).
6. Подключите SSL и включите HTTPS-редирект в `.htaccess`.
7. После согласования перенесите файлы на основной домен.

## Что проверить до публичного запуска
- Точные названия и адреса объектов.
- Состав работ и даты по каждому проекту.
- Актуальные лицензии и выписки из реестров.
- Актуальность телефона, почты и режима работы.
- Юридическую редакцию политики конфиденциальности.
- Право публикации всех фотографий.
